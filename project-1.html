AI Chatbot (Vortex AI)

AI (Artificial Intelegent) Yang menggunakan API Key Google gemini dengan UI Yang lebih baik dan responsive

Kunjungi

Penjelasan Projek:

Proyek ini adalah aplikasi kecerdasan buatan (AI) yang memanfaatkan model Gemini 2.5 Flash dari Google untuk menganalisis sentimen dari review pelanggan, membuat chatbot yang cerdas, merangkum dokumen panjang. Dibangun dengan library google-generativeai, proyek ini menunjukkan bagaimana model AI generatif terbaru dapat diintegrasikan untuk menyelesaikan tugas-tugas kompleks dengan cepat dan efisien.

Bahasa program yang digunakan:

HTML
CSS
JavaScript
